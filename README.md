# Desafio-sql

## Dependências

- Python
- Pip
- Flask
- SqlAlchemy

## Como executar:

Clone o projeto

>git clone https://github.com/Baquara/Desafio-sql

>cd Desafio-sql

Instale as dependências:

>pip install -r requirements.txt

Execute o projeto:

>python procedures.py

Abra a URL no navegador:

> http://127.0.0.1:5000/
